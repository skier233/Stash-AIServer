# Stash AI Server

Backend server for Stash providing AI-powered features, task management, and extensible plugins.

## Quick Start

See the [installation instructions](https://github.com/skier233/Stash-AIServer/wiki/AI-Overhaul-Installation-Instructions) to get started

## Docker Compose

You can start the production backend directly with:

```bash
docker compose up -d
```

To run the development environment (requires source code):

```bash
docker compose --profile dev up
```
